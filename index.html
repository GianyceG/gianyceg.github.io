ARIMA Model

The ARIMA model is defined by the parameters \( (p, d, q) \), where:

  • \( p\) is the number of lag observations (autoregressive terms).
  • \( d\) is the number of times the data must be differenced to make it stationary.
  • \( q\) is the number of lagged forecast errors (moving average terms).

    • My ARIMA(5,0,5) Model:

    For this specific COVID-19 data and the model that I will end up generating, the following is the mathematical expression of my ARIMA model in general terms:

    \[Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \dots + \phi_5 Y_{t-5} + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \dots + \theta_5 \epsilon_{t-5} + \epsilon_t\]

    Where:

    • \( Y_t \): The value of the time series at time \( t \) (in this case, the number of COVID-19 confirmed cases in Florida).
    • \( c \): Constant term (intercept).
    • \( \phi_i \) (\( i = 1, 2, \dots, 5 \)): Coefficients for the Auto-Regressive (AR) terms (from lag 1 to lag 5).
    • \( Y_{t-i} \): The value of the time series at previous time steps (\( t-1, t-2, \dots, t-5 \)).
    • \( \theta_i \) (\( i = 1, 2, \dots, 5 \)): Coefficients for the Moving Average (MA) terms (past errors).
    • \( \epsilon_{t-i} \): Errors at previous time steps (\( t-1, t-2, \dots, t-5 \)).
    • \( \epsilon_t \): Error term at time \( t \).

ARIMA Model Application to Python

Libraries Used for ARIMA

To perform time series forecasting using ARIMA, the following key Python libraries were utilized:

  • Pandas: Used for handling and manipulating the dataset, particularly converting date columns and preparing it for time series analysis.
    • pandas.read_csv() to load the dataset.
    • pandas.to_datetime() to convert the date column to a format usable for time series.
    • pandas.DataFrame.diff() to apply differencing for stationarity.
  • Matplotlib: For visualizing the time series, differencing, and residual diagnostics.
    • matplotlib.pyplot.plot() to plot time series, ACF, PACF, and residuals.
  • Statsmodels: Provides the implementation of the ARIMA model and time series diagnostics like ACF and PACF.
    • statsmodels.tsa.arima.model.ARIMA for fitting the ARIMA model.
    • statsmodels.graphics.tsaplots.plot_acf() to plot the autocorrelation function (ACF).
    • statsmodels.graphics.tsaplots.plot_pacf() to plot the partial autocorrelation function (PACF).

Process

The process for the ARIMA model involved several key steps:

  1. Applied first-order differencing to the time series data to make it stationary.
  2. Utilized the ACF and PACF plots to determine the appropriate model parameters (p, d, q).
  3. Fitted an ARIMA model based on the selected parameters.
  4. Checked the residuals of the model to ensure that they resemble white noise, indicating a good fit.
